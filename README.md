# claude-skills

Claude에 설치된 스킬을 관리하는 스킬 모음입니다.

---

## 📦 skill-management

설치된 Claude 스킬 전체를 관리하는 통합 관리 스킬입니다.

### 기능

| 모드 | 트리거 | 설명 |
|------|--------|------|
| 카탈로그 | "내 스킬 보여줘", "스킬 목록" | 설치된 스킬을 카테고리별로 정리해서 보여줍니다 |
| 중복 감지 | "스킬 중복 확인", "겹치는 스킬 있어" | 비슷한 기능의 스킬을 찾아 분석합니다 |
| 스킬 추천 | "어떤 스킬 써야 해?", "스킬 추천" | 작업에 맞는 스킬을 추천합니다 |
| 스킬 진단 | "스킬 진단", "스킬 점수 매겨줘" | 각 스킬의 품질을 점수로 평가합니다 |

### 특징

- 스킬 추가/삭제 시 자동 반영 (하드코딩 없음)
- 설치된 스킬 구성에 따라 카테고리 자동 생성
- 기존 스킬 수정 없이 분석만 수행

### 설치

```bash
npx skills add paran-coder/claude-skills@skill-management
```

### 사용 예시

```
"내 스킬 보여줘"
"랜딩페이지 만들려는데 어떤 스킬 써야 해?"
"스킬 중복 확인해줘"
"스킬 진단해줘"
```
